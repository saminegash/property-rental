# Design Reference

## Visual Direction
The platform follows the "MyProperties" design language, characterized by a clean, modern, and professional aesthetic suitable for a high-trust marketplace.

### Color Palette
- **Primary:** Clean White and Blue.
- **Backgrounds:** Primarily white and light surface colors for contrast.
- **Accents:** Blue for primary actions, links, and highlights.

### Core UI Components
- **Hero Section:** Strong, commanding hero section on the homepage to establish brand identity and immediate value proposition.
- **Search:** Large, highly visible search box front-and-center in the hero section to drive user engagement.
- **Cards:** Content (properties, cars) is displayed in rounded cards.
- **Shadows:** Soft, subtle shadows on cards and floating elements to create depth without clutter.
- **Badges:** Prominent use of "Verified" and "Featured" badges on listing cards to build trust.
- **Trust Elements:** Dedicated sections highlighting platform safety, admin verification, and user reviews.

### Layout Principles
- **Mobile-First:** All layouts must be responsive, prioritizing the mobile experience while scaling elegantly to desktop.
- **Property-First Homepage:** The root URL (`/`) is dedicated to properties.
- **Cars Module:** The `/cars` route maintains the same visual identity but is functionally distinct.

### Media
- **Galleries:** Listings feature high-quality image galleries (5-10 images required per listing) with clean carousel or grid layouts.
